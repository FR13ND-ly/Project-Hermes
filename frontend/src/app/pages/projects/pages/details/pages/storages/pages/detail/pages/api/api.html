API Integration — Connecting External Applications

Buckets are private. External applications access files using the bucket's app_id / secret_key pair, automatically published to the project environment pool.

Bucket Credentials

Access is done with the app_id / secret_key pair, automatically published to the project environment pool as BUCKET_<slug>_APP_ID and BUCKET_<slug>_SECRET_KEY. You can regenerate it from the "Settings" tab → "Rotate Credentials".
